# Keyboard Shortcuts

Master these shortcuts to work faster in Astonish Studio.

## General

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + S** | Save flow |
| **Cmd/Ctrl + Z** | Undo |
| **Cmd/Ctrl + Shift + Z** | Redo |
| **N** | New flow |
| **Escape** | Close panel / Deselect |

## Canvas Navigation

| Shortcut | Action |
|----------|--------|
| **Scroll** | Zoom in/out |
| **Click + Drag** (background) | Pan canvas |
| **Cmd/Ctrl + 0** | Reset zoom |
| **Cmd/Ctrl + 1** | Fit to screen |

## Node Operations

| Shortcut | Action |
|----------|--------|
| **Click** | Select node |
| **Delete / Backspace** | Delete selected node |
| **Cmd/Ctrl + D** | Duplicate node |
| **Arrow keys** | Move selected node |

## Edge Operations

| Shortcut | Action |
|----------|--------|
| **Click edge** | Select edge |
| **Delete / Backspace** | Delete selected edge |

## Running

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + Enter** | Run flow |
| **Escape** | Stop execution |

## YAML Editor

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + Shift + Y** | Toggle YAML view |

## Tips

- Most shortcuts work when the canvas is focused
- Click the canvas background to ensure focus
- Shortcuts don't work when editing text fields
