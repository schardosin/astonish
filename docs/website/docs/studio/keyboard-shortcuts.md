# Keyboard Shortcuts

Quick reference for Studio keyboard shortcuts.

## Global

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Command palette |
| `Ctrl+Shift+D` | Toggle debug panel |
| `Ctrl+,` | Open Settings |
| `Ctrl+N` | New chat session |
| `Ctrl+Tab` | Switch between tabs |

## Chat

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in input |
| `Up` / `Down` | Navigate message history |
| `Escape` | Cancel current generation |
| `/` | Open slash command menu |
| `Ctrl+L` | Clear display |

## Flow Editor

| Shortcut | Action |
|----------|--------|
| `F5` | Run flow |
| `Ctrl+G` | AI Assist (generate nodes) |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Delete` | Remove selected node/connection |
| `Ctrl+A` | Select all nodes |
| `Ctrl+D` | Duplicate selected node |
| `Space` (hold) | Pan canvas (drag) |
| `Ctrl+Plus` | Zoom in |
| `Ctrl+Minus` | Zoom out |
| `Ctrl+0` | Reset zoom |

## Fleet

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+F` | Open Fleet tab |
| `Ctrl+Enter` | Approve pending action |
