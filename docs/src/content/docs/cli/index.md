Astonish provides 16 CLI commands covering chat, session management, flows, daemon, channels, fleet, and more.

## Global Options

| Flag | Description |
|------|-------------|
| `-h`, `--help` | Show help for any command |
| `-v`, `--version` | Print the Astonish version |

## Command Reference

| Command | Description |
|---------|-------------|
| `chat` | Start an interactive chat session with AI |
| `sessions` | Manage and inspect persistent sessions |
| `flows` | Design and run AI automation flows |
| `studio` | Launch the visual Studio web UI |
| `daemon` | Manage the background service |
| `channels` | Manage Telegram and Email channels |
| `scheduler` | Manage scheduled jobs |
| `fleet` | Manage fleet plans and agent teams |
| `credential` | Manage the encrypted credential store |
| `skills` | Manage CLI tool skill guides |
| `memory` | Manage semantic memory and knowledge |
| `tools` | Manage MCP tools |
| `tap` | Manage extension repositories |
| `config` | Manage configuration files |
| `setup` | Run interactive setup wizard |
| `demo` | Generate animated HTML terminal demos |

Most commands have subcommands. Use `astonish <command> --help` for details.
