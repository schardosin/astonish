## The Astonish Experience

### Talk to your AI assistant

Start a conversation in the Studio web UI or the CLI. Ask it to read files, run commands, search the web, manage infrastructure — anything. The agent picks the right tools, asks for confirmation, and gets it done.

![Studio Chat — conversational AI with tool use](/astonish/images/experience-chat.webp)

### It remembers everything

Astonish maintains a persistent memory across all sessions. Infrastructure details, credentials, preferences, past solutions — the agent recalls relevant knowledge automatically so you never repeat yourself.

![Memory recall — the agent uses knowledge from previous sessions](/astonish/images/experience-memory.webp)

### Chat from anywhere

Connect Telegram or Email as communication channels. Chat with your assistant from your phone, receive scheduled task results, and manage fleet sessions — all from the same bot.

![Telegram integration — chat with Astonish from your phone](/astonish/images/experience-telegram.webp)

### Automate what works

Had a successful chat session? Distill it into a reusable flow with one command. Schedule it to run on a cron, broadcast results to Telegram, and never do that task manually again.

![Flow distillation — convert chat sessions into repeatable automations](/astonish/images/experience-distill.webp)
