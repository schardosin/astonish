## [2026-07-10] Task 2 implementation notes

- **Interface, not concrete type.** `pkg/provider/resolve.go` defines `UserDefaultSettings interface { GetDefaultProvider() string; GetDefaultModel() string }` — narrow, two-method contract. This is what let the RED tests use `fakePersonalSettings` (no `pkg/store` import). Todo 3's `store.PersonalSettings` will satisfy it structurally with `GetDefaultProvider()` / `GetDefaultModel()` getters.
- **Nil safety: two layers.** `ApplyUserDefault` guards `if us == nil { return cfg }` for the untyped-nil path. The typed-nil path (`(*fakePersonalSettings)(nil)` boxed into a non-nil interface) is handled by the concrete type's nil-receiver checks on `GetDefaultProvider`/`GetDefaultModel` returning `""` — which the empty-string branches then no-op through. Both cases covered without `reflect`. Todo 3's `store.PersonalSettings` getters MUST also implement nil-receiver safety (return `""`) to preserve this contract at real call sites.
- **Mutate-in-place, return-self.** Both overlays mutate `cfg.General.*` directly and return the same pointer. Test `if result != tc.cfg` catches any accidental copy. Enables chainable call sites: `cfg = ApplyUserDefault(cfg, us); cfg = ApplyProviderOverride(cfg, p, m)`.
- **Providers map untouched.** Verified by `wantProvidersLen` assertions across 6 subtests plus the `ChainedCascadeOrder` test (3 providers survive all overlays). The overlays are default-only; the additive Providers map remains the sole responsibility of `applyProviderLayer`.
- **Empty-string sentinel semantics.** Both overlays treat `""` as "inherit from cascade below" — perfectly mirrors the existing `applyProviderLayer` behavior at line 65-70. This is the "unpin"/"clear" contract for `ApplyProviderOverride`: passing `("", "")` restores whatever the lower layers produced.
- **Placement.** Public exported funcs added between `ResolveEffectiveConfig` and the two private `apply*` helpers. Interface defined immediately above its consumer (Go idiom: interface near use). File grew 95 → 139 LOC; still well under the 250 ceiling.
